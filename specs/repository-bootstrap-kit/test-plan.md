# 仓库启动包测试计划

## 状态

`active`

## 目的

定义 `FR-001` 第一版实现后的验证方式。

## 测试范围

- 目录初始化行为
- 文档模板生成行为
- 变量替换行为
- 覆盖保护行为

## 测试用例

| ID | 用例 | 目标 |
| --- | --- | --- |
| `TC-001` | 在空目录初始化仓库 | 验证能生成标准治理骨架 |
| `TC-002` | 提供项目名称和简介 | 验证模板变量替换正确 |
| `TC-003` | 目标目录已有关键文件 | 验证不会静默覆盖 |
| `TC-004` | 初始化完成后按 README 路径检查文档 | 验证入口链路完整 |

## 验证方式

- 自动化检查生成出的目录和文件存在性
- 抽样检查模板变量是否被正确替换
- 检查输出结果中是否包含 README、doc-map、workflow、PLAN-001 等关键文档

## 通过标准

- 所有关键文件生成成功
- 占位符未残留在最终文档中
- 覆盖保护符合预期
- 初始化后的仓库可以直接进入下一步计划或需求工作
